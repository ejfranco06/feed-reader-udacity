frontend-nanodegree-feedreader
================================
A feed reader that will take provide a feed for blog post on
* Udacity
* CSS Tricks
* html5rocks
* Linear Digressions

How to start
============
* Download or clone the project

* Open the index.html file on your favorite browser
  ```
  open index.html
  ```

How to use
===========
* Click on the menu button on the left to select a topic

* Click on an entry on screen to read more about it
